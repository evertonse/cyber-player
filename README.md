
# Start
Currently we only support compiling for windows from linux.
So mingw toolchain has to be available.

```bash
    gcc nob.c -o nob && ./nob mingw sdl3 run
```

# Notes

This project uses a multiple places modified version of an older raylib. Do not update it.
